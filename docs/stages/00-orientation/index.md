# 00 Orientation

**Phase 1 - Foundations**  
Stage 0 of 14. Start here if you are new to AI agents or want the full path.

## Goal

Understand what AI agents are, what this roadmap covers, and how to use the learning database.

## Learn

- [What Are AI Agents?](what-are-ai-agents/index.md)
- How this roadmap is organized
- How to choose the right starting point
- What counts as real progress in each stage

## Build

Create a short learning log that answers three questions:

- What kind of AI agent do you want to build?
- Which skills do you already have?
- Which stage should you start from?

## Exit Criteria

- You can explain the difference between a chatbot, workflow, and agent.
- You understand that agents combine LLMs, tools, state, and feedback loops.
- You know how to navigate the staged roadmap.
