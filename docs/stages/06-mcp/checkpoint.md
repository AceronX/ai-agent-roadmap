# Checkpoint: MCP

- [ ] I can explain MCP hosts, clients, and servers.
- [ ] I can describe MCP as agent-to-tool communication.
- [ ] I can identify what tools and resources an MCP server exposes.
- [ ] I can explain why tools are actions and resources are readable context.
- [ ] I can explain local vs remote MCP tradeoffs.
- [ ] I built or sketched a minimal MCP server.
