# Reference

Shared material that applies across many stages. Use it whenever you need a definition, a good resource, or a reusable checklist.

- **[Glossary](glossary.md)** — plain definitions for agent, tool, MCP, A2A, RAG, memory, and more
- **[Resources](resources/index.md)** — curated papers, courses, tools, frameworks, and communities
- **[Best Practices](best-practices/index.md)** — guidance for agent design, tools, evaluation, security, and production

## How to Add Material

Add broad material here only when it applies to many stages. If a resource belongs to one topic, add it to that topic folder instead.

Good additions are:

- Official docs and specifications
- High-quality papers and courses
- Practical implementation guides
- Tools with clear use cases
- Example repositories with working code
